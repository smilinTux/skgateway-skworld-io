Core Capabilities

Everything you need to secure, monitor, and manage AI inference traffic

🛡️

Policy Engine

Declarative YAML policies for routing, rate limiting, content filtering, and access control. 12 policy rule types with real-time enforcement.

📊

SIEM Integration

Native CEF event format. Drop-in integration with Splunk, Elastic, and rsyslog. Full audit trail for every AI request and response.

🔌

Multi-Provider Support

Route to OpenAI, Anthropic, Ollama, NVIDIA, and any OpenAI-compatible API. Automatic failover and load balancing across providers.

⚡

WebSocket Streaming

Real-time streaming proxy with token-level monitoring. Stream responses to clients while simultaneously auditing every token.

🎨

SOC Dashboard

Real-time security operations dashboard with soul-aligned color schemes. Monitor traffic, threats, and policy violations at a glance.

🔗

OpenClaw & Claude Code

First-class integration with OpenClaw agent framework and Claude Code. Drop-in proxy for any AI-powered development workflow.

Quick Start

Up and running in minutes

$ git clone https://github.com/smilinTux/skgateway.git
$ cd skgateway
$ npm install
$ cp config/default.yaml config/local.yaml
$ npm start
# Gateway running on :3100
